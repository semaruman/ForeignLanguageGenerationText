<p>У каждого человека есть набор слов, которые он знает на иностранном языке. В приложение добавляются изученные слова, а оно генерирует текст, который пользователь полностью понимает. схема изучения языка такая:</p>
<ol>
  <li>Изучил новые слова</li>
  <li>Добавил слова в приложение</li>
  <li>Прочитал и перевёл несколько текстов, которые включают все изученные слова</li>
</ol>
